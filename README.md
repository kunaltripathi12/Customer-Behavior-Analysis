# Customer-Behavior-Analysis
An end-to-end data analysis project focused on cleaning and analyzing customer transaction data using Python, SQL, and Power BI. The project involved data preprocessing, customer segmentation, and dashboard development to identify purchasing patterns and support data-driven decision-making.

📊 Customer Behavior Analysis

📌 Overview

This project analyzes customer purchasing behavior using Python, SQL, and Power BI. The objective was to clean raw transaction data, identify trends, and generate actionable insights to support business decisions.

 🎯 Business Problem

The dataset contained inconsistent, incomplete, and unstructured transaction records, making it difficult to understand customer behavior and product performance.

 🛠 Tools Used

* Python (Pandas, NumPy)
* SQL (Joins, GROUP BY, CTEs, Window Functions)
* Power BI (Dashboard, KPIs, Filters)


🔄 Workflow

1. Data Cleaning & Preparation

* Cleaned 5,000+ transaction records
* Handled missing values and inconsistencies using Pandas
* Removed duplicates and standardized formats
* Ensured data accuracy and completeness

2. Data Analysis (SQL)

* Performed customer segmentation
* Analyzed purchasing patterns and product performance
* Used GROUP BY, CTEs, and window functions

3.Dashboard (Power BI)

* Built interactive dashboard with KPIs:

  * Total Revenue
  * Top Products
  * Customer Segments
  * Sales Trends
  * Added filters for dynamic analysis


📈 Key Insights

* Identified top-performing products contributing to revenue
* Found high-value customer segments
* Discovered purchasing trends across categories
* Improved reporting efficiency by **30%**


✅ Outcome

* Converted raw data into a structured, analysis-ready dataset
* Delivered clear insights through dashboards
* Enabled data-driven decision-making
